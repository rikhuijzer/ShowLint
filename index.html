ShowLint

This website shows linting results for patterns on repositories by using Comby. As an example, lets run the pattern p1 on this repository. The pattern finds toml files containing name = "..." and adds a comment before that line:

SystemError: opening file "/home/runner/work/ShowLint/ShowLint/configs/p1.toml": No such file or directory

The output is

failed process: Process(podman run --volume /home/runner/work/ShowLint/ShowLint:/project --rm --workdir /project -it comby/comby -config configs/p1.toml -file-extensions toml, ProcessExited(1)) [1]
