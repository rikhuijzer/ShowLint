ShowLint

This website shows linting results for patterns on repositories by using Comby. As an example, lets run the pattern p1 on this repository. The pattern finds toml files containing name = "..." and adds a comment before that line:

[p1]

match='name = ":[name]"' 

rewrite='''
# A new comment.
name = ":[name]"'''

The output is

IOError: could not spawn comby -config configs/p1.toml -f toml: no such file or directory (ENOENT)
