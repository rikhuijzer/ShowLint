Patterns

This page lists the patterns as defined in src/patterns.jl.

1: Test: Add comment in Project.toml

tags: test, julia

A test pattern to see if things work. 

match='name = ":[name]"' 

rewrite='''
# A new comment.
name = ":[name]"'''

2: Avoid var === nothing

tags: julia

Note that ismissing can be slower, see https://github.com/JuliaLang/julia/issues/27681. 

match='if :[var] === nothing' 

rewrite='if isnothing(:[var])'

3: Avoid var !== nothing

tags: julia

See pattern 2. 

match='if :[var] !== nothing'

rewrite='if !isnothing(:[var])'

4: Replace Array{T,1} with Vector{T}

tags: julia

Vector{T} is an alias for Array{T,1}. 

match='Array{:[T],1}'

rewrite='Vector{:[T]}'
