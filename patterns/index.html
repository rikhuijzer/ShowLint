Patterns

This page lists the patterns as defined in src/patterns.jl.

1: Prefix project name with Test.

tags: test, julia

A test pattern. 

match='name = ":[name]"' 

rewrite='''
name = "Test:[name]"'''

2: Use ismissing instead of === and !==

tags: julia

Note that ismissing can be slower, see https://github.com/JuliaLang/julia/issues/27681. 

match=':[var~[^;]] :[neg~[=|!]]== missing' 

rule='where rewrite :[neg] { "=" -> "" }'

rewrite=':[neg]ismissing(:[var])'

3: Replace Array{T,1} with Vector{T}

tags: julia

Vector{T} and AbstractVector{T} are aliases for respectively Array{T,1} and AbstractArray{T,1}. 

match='Array{:[T],1}'

rewrite='Vector{:[T]}'
